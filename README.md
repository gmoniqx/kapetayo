# KapeTayo

KapeTayo is a React + TypeScript web app for discovering cafés in Marikina City.

## Development

Requirements:

- Node.js 18+
- npm

Install and run:

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` — start local dev server
- `npm run build` — create production build
- `npm run preview` — preview production build
- `npm run lint` — run ESLint
- `npm run test` — run tests once

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
